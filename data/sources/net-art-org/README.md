# net-art.org Source Notes

Source: net-art.org / AND (Artists Network Database).

Files captured here:
- `raw/artist-listing.html` — artist listing landing page
- `raw/sitemap.xml` — public sitemap of crawlable site URLs
- `artworks.json` — first-pass extraction of sitemap URLs as site-page records

Notes:
- The public artist-listing page is thin in static HTML, but the sitemap exposes the site's crawlable URL set.
- This first pass is page-oriented rather than artwork-oriented.
- Useful as a discovery/index source for later focused crawling.

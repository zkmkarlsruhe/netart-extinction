# Goldsen Source Notes

Source: Rose Goldsen Archive of New Media Art, Cornell University Library.

Files captured here:
- `raw/internet.html` — Internet Art landing page
- `raw/online.html` — Online Journals, Net Art, Listservs holdings page
- `artworks.json` — first-pass extracted collection/holding records from visible page text

Notes:
- The Goldsen site is browseable HTML, but much of the archive is collection-oriented rather than a clean artwork-by-artwork database.
- Some holdings are explicitly marked as archived offline.
- This first pass is best treated as a collection registry for later manual or scripted expansion.

# netart latino Source Notes

Source: netart latino database.

Files captured here:
- `raw/landing.html` — main database page
- `artworks.json` — first-pass extraction of external links grouped by nearest section anchor on the page

Notes:
- This source is not a clean modern API. It behaves more like a curated link database.
- The extracted JSON is link-oriented and should be treated as discovery data for later normalization.
- The `section` field reflects the nearest anchor/section identifier in the page, which often maps to country or topical grouping.
